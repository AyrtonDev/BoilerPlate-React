# Installation

<br />

First you need to give git clone!

<br />

#### GitHub Project

<br />

```sh
$ git clone https://github.com/AyrtonDev/boilerplate-react.git
```

<br />

#### After the download project, git in the paste and install dependecies

<br />

##### NPM

```sh
$ cd boilerplate-react | npm
```

<br />

##### YARN

```sh
$ cd boilerplate-react | yarn
```

<br />

### Commands

<br />

#### For run development

```sh
$ npm run dev
```

<br />

#### For run test mode watch

```sh
$ npm run watch
```

<br />

#### For run storybook

```sh
$ npm run storybook
```
